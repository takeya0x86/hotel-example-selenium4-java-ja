### 概要

#### プログラミング言語

* Java

#### 自動化フレームワーク

* [Selenium WebDriver](https://www.selenium.dev/)

#### テスティングフレームワーク

* [JUnit 5](https://junit.org/junit5/)

* JDK 21
* Google Chrome

##### Windows


##### macOS/Linux

```
./gradlew clean test
```
